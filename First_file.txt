Git commands:
The basic Git syntax is program | action | destination.

    git add <filename> - add file to staging area
    git commit - commits files in staging area to local branch
    git commit -m "comments" - commits with comments
    git push <where from> <to where> - pushes the local git repository to specified server branch

    git status - see the commit status of of the current git repository
    git log - shows the log of activities in the branch (q to quit)
